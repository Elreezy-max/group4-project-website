# 🚀 HirePath

HirePath is a frontend-only hiring platform built using HTML, CSS, and JavaScript.
It simulates a real-world job portal with multiple user roles and workflows.

---

## ✨ Features

* 👤 Applicant dashboard (search, apply, save jobs)
* 🏢 Employer panel (post jobs, manage candidates)
* 🛠️ Admin controls (approve jobs and users)
* 📊 Realistic UI flows (interviews, messaging, AI matching demo)
* 💾 LocalStorage-based authentication (demo purpose)
